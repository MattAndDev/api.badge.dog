## Options (defaults)
```js
// all size in pixels
// all values need to be url-encoded
{
  title: 'badge.dog',  // svg title
  leftText: 'BADGE',
  leftBgColor: '#20A69A',
  leftTextColor: '#FFF',
  rightText: 'DOG',
  rightBgColor: '#043B40',
  rightTextColor: '#FFF',
  paddingVer: 16,
  paddingHor: 16,
  fontSize: 14,
  googleFontName: 'Lato' // as shown in css import url
}
```
